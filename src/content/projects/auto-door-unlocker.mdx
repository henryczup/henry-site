## What it does
An ESP32-based system that detects apartment intercom calls and automatically sends the correct DTMF sequence to unlock the building entrance.

## Highlights
- Detects incoming intercom calls using ring detection
- Initiates an off-hook response and DTMF unlock sequence
- Integrates ESP32 GPIO control with a DTMF module
- Turns a repetitive apartment-entry workflow into a small embedded automation
