## What it does
A grade distribution platform for UW-La Crosse that helps students compare historical course outcomes before registration.

## Highlights
- Analyzed 17,000+ historical grades
- Built search and filtering tools around course and grade data
- Reached 2,800 users and 107,000+ site events
- Averaged 3m 53s engagement per active user
- Monetized registration-period traffic through ad placement
